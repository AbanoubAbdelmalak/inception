= General Settings

.General settings
[cols="4*", options="header"]
|===
| Setting
| Description
| Default
| Example

| warnings.unsupportedBrowser
| Warn about unsupported browser
| true
| false

| debug.showExceptionPage
| Show a page with a stack trace instead of an "Internal error" page. Do not use in production!
| false
| true

| login.message
| Custom message to appear on the login page, such as project web-site, annotation guideline link, ... The message can be an HTML content.
| _unset_
| `<span style="color:red; font-size: 200%;">Use are your own risk.</span>`

| user.profile.accessible
| Whether regular users can access their own profile to change their password and other profile information. This setting has no effect if WebAnno is running in pre-authentication mode.
| false
| true

| userSelection.hideUsers
| Whether the system's usernames will be shown into the dropdown list at the Users tab. In order for the Project manager to add a user into an annotation project she has to write into the same text Field the annotator’s full username. When there is a full match this username (and only this) is displayed in the dropdown list where the Project Manager can select it and add it to her project.
| false
| true

| userSelection.minLengthCharacters
| The number of characters the Project Manager has to type in order to see the dropdown list at the Users tab. This setting has no effect if userSelection.hideUsers=false.
| 2
| 4
|===
